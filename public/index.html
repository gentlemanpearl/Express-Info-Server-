Express Info Server

Use the API endpoints described below.

Routes

  • / - Welcome message.
  • /about - About this project.
  • /contact - Contact info (served from public/contact.html).
  • /api/users - JSON list of users.
  • /weather/:city - Example weather JSON.

Try it

You can add a new user with POST to /api/users (JSON body: { "name": "Your Name" }).
